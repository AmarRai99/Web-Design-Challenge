Summary: Latitude vs. X

The purpose of this project is to analyze the weather changes closer to the equator. To accomplish this, the data of over 500 cities have been pulled from the OpenWeatherMap API.

Matplotlib is used to plot the various aspects of Weather vs. Latitude. The factors that are considered include Maximum Temperature, Cloudiness, Wind Speed, and Humidity. This site provides the data source and visualizations created as part of the analysis, as well as any explanations and descriptions for any trends and correlations that have been obtained.
